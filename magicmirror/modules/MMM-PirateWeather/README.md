# MMM-PirateWeather

MagicMirror²-Modul für die Pirate Weather API (DarkSky-kompatibel).

## Installation

```bash
npm install node-fetch@2
```

## Konfiguration

```js
{
  module: "MMM-PirateWeather",
  position: "top_right",
  config: {
    apiKey: "<DEIN_PIRATE_KEY>",
    lat: 36.721,
    lon: -4.421
  }
}
```
